# Proyecto Compraventa (Java)

## Descripción
Este proyecto es una aplicación en Java que simula un sistema de compraventa de productos (videojuegos y merchandising). Se maneja mediante un menú por consola donde el usuario puede elegir distintas opciones.

## Funcionalidades
- Ver todas las compras
- Buscar una compra por ID
- Mostrar una compra aleatoria
- Ver información de merchandising
- Salir del programa

## Estructura
El proyecto está organizado con varias clases:
- Producto (clase base)
- Videojuego y Merchandising (heredan de Producto)
- Compra
- Usuario
- Tienda
- HistorialPrecios

También se usa una clase ComprasDAO para gestionar los datos.

## Conceptos usados
- Programación orientada a objetos
- Herencia
- Uso de listas
- Menús con switch

## Ejecución
Ejecutar la clase `App` y seguir las opciones del menú en consola.
